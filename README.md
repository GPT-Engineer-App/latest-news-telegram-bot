# latest-news-telegram-bot

Code for a Telegram bot that will parse news from https://quote.rbc.ru/?utm_source=topline and publish in the Telegram channel only the latest or newest news on the RBC website page.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/latest-news-telegram-bot.git
cd latest-news-telegram-bot
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
